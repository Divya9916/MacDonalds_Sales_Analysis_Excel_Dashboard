# MacDonald-s-Restaurant-Sales-Analysis
McDonald’s Restaurant Sales Analysis
Analyzed McDonald’s sales data in Excel by cleaning, transforming, and exploring order patterns, product performance, and revenue trends. Used Pivot Tables, time-based analysis, and calculated metrics to uncover insights on top-selling items, peak hours, and category-wise revenue for data-driven decision-making.

## Project Overview
This project analyzes McDonald’s restaurant sales dataset to extract actionable insights that support business decisions. The dataset includes transaction-level data such as order time, product category, quantity, and sales.
Using Microsoft Excel, the data was cleaned, transformed, and analyzed to identify trends in customer behavior, sales performance, and operational efficiency.

## Tools & Technologies
Microsoft Excel
Pivot Tables & Charts
Data Cleaning & Transformation
Time-based Feature Engineering

## Dataset Description
Column	Description
Order Date	Date of the order
Order Time	Time of the order
Item Name	Product purchased
Category	Product category (Burger, etc.)
Quantity	Number of items sold
Sales	Revenue generated from the item
Note: Dataset was pre-processed for missing values and consistency.
Data Cleaning & Preprocessing
Removed duplicate records to ensure accuracy
Handled missing/null values using Excel methods
Converted time from AM/PM to 24-hour format
Created time slots (Morning, Afternoon, Evening, Night)
Standardized category and product naming

## Analysis & Insights
Sales Analysis: Identified overall revenue and order trends
Category Performance: Compared revenue contribution across categories
Product Analysis: Identified top-selling and high-revenue items
Time-Based Analysis: Analyzed peak ordering hours
Customer Behavior: Studied weekday vs weekend patterns

## Key Findings
Burgers generate the highest revenue among all categories
Side Salad is the most frequently ordered item
Peak sales occur during afternoon and evening hours
Weekdays contribute ~71.7% of total sales
High-frequency items are not always high-revenue items
